HearSay is installed

The side panel has Screen Reader Lab and Dictionary — the same tools as the HearSay website, for use alongside whatever course site or document you have open.

  1. Pin HearSay — puzzle icon → pin HearSay.
  2. Open your course page in a normal tab (LMS, syllabus, quiz, lab, etc.).
  3. Click the HearSay icon — the side panel opens on the right.
  4. ☁ Connect once (Supabase URL + anon key).
  5. Screen Reader Lab — select text on the page → Pull from page → hear raw vs dictionary speech.
  6. Dictionary tab — edit terms and download the NVDA add-on for students.
