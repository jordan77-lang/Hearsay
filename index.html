HearSay

I can speak it — and spell it — for screen readers.

Internal tool for course authors. Paste curriculum, hear how it reads, copy Canvas HTML so students don’t need to edit their own screen-reader dictionaries.

Get started

Open HearSay in your browser Paste, analyze, hear, copy Canvas HTML — no install Download Chrome extension Side panel + pull selection from Canvas

Install Chrome extension

  1. Download hearsay-chrome-extension.zip (run npm run pack:extension before deploying if the file is missing).
  2. Extract the zip to a folder you will keep (e.g. Documents/HearSay-extension).
  3. Open chrome://extensions, turn on Developer mode.
  4. Click Load unpacked and select the extracted folder (must contain manifest.json).
  5. Click the HearSay toolbar icon to open the side panel. Use Pull selection from page while authoring in Canvas.

Class dictionaries

After opening the tool, click ☁ Connect and sign in with your Supabase URL and anon key. Pick your class in the dictionary panel to load its pronunciation rules.
