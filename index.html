HearSay

Speak it clearly — for every screen reader.

Tools for course authors: build class dictionaries, test quiz speech, and export screen reader dictionaries for students — without asking them to edit personal dictionaries.

Install Chrome extension

  1. Download hearsay-chrome-extension.zip.
  2. Extract the zip to a folder you will keep (for example Documents/HearSay-extension). The folder must contain manifest.json at the top level.
  3. Open chrome://extensions and turn on Developer mode.
  4. Click Load unpacked and select the extracted folder (not the .zip file).
  5. Pin HearSay from the puzzle icon, then click the HearSay toolbar icon to open the side panel. Use ☁ Connect once, then Screen Reader Lab or Dictionary. On a course page, select text and choose Pull from page (Lab tab).
