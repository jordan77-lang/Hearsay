HearSay screen-reader listening test

How to run this test (Windows + NVDA, free):

  1. Install NVDA from nvaccess.org and start it (Ctrl+Alt+N).
  2. Open this page in Chrome or Firefox.
  3. Click the heading, then press the Down arrow to read one line at a time.
  4. For each example, compare what you hear across variants 1–4.
  5. Variant 1 (Raw) should sound wrong; 2–4 should sound correct.

NVDA tip: press NVDA+Down to "say all", or NVDA+Numpad Plus from the top. For MathML, NVDA 2026.1+ reads it automatically (MathCAT built in).
